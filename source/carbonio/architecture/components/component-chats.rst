Overview
--------

The **Chats** component is responsible for managing chat services within the Carbonio platform.

It encompasses the components and functionalities that facilitate real-time communication and collaboration among users through chat, ensuring seamless message delivery, notifications, and integration with other platform services.


Packages
--------

The following packages are required for the **Chats** component:

.. code-block:: bash

   carbonio-message-dispatcher
   carbonio-ws-collaboration
   carbonio-push-connector
   carbonio-notification-push


.. important::

   Installation procedures are documented in the dedicated :ref:`install` section.
   This page contains only the component description and package list.
