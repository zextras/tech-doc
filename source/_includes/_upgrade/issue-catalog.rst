.. card:: New package for 24.12.X

   Regardless of the version from which you are upgrading, the
   **24.12.X** requires that the package ``carbonio-catalog`` be
   installed on the Node installing the **Proxy Role**.

   To install it, execute the following command.

   .. tab-set::

      .. tab-item:: Ubuntu
         :sync: ubuntu

         .. code:: console

            # apt install carbonio-catalog

      .. tab-item:: RHEL
         :sync: rhel

         .. code:: console

            # dnf install carbonio-catalog
